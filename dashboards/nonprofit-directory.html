Data Source

Organization data is sourced from the ProPublica Nonprofit Explorer, which aggregates IRS Form 990 filings for tax-exempt organizations. Financial data is updated as new filings become available (typically 6-18 months after fiscal year end).

Source ProPublica Nonprofit Explorer API Data IRS Form 990, 990-EZ filings Coverage All registered 501(c)(3) organizations
