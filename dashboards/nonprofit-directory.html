Data Sources

990

IRS Form 990

Annual information return filed by tax-exempt organizations. Includes revenue, expenses, assets, liabilities, compensation, and program service accomplishments.

Annual filings 6-18 month lag
PP

ProPublica Nonprofit Explorer

Aggregates IRS Exempt Organizations Business Master File and annual financial extracts into a searchable API covering every registered 501(c)(3) organization.

Live API All 501(c)(3) orgs
BMF

IRS EO Business Master File

The authoritative registry of tax-exempt organizations — provides EIN, name, address, NTEE classification, ruling date, and current exempt status.

Monthly updates Organization registry
